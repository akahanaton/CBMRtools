GeneSet

Description

Class GeneSet represents a compendium (list) of genesets, each representing a set of gene identifiers

Slots

source.file
the file (with path) where the geneset originates from

geneset
the list of character vectors each representing a set of gene IDs

type
type of gene ID (gene symbol, ensemble ID, entrez ID, etc.)

verbose
verbose logging of class operations

do.save
undefined

name
of geneset compendium (most often the source.file w/o the path)
