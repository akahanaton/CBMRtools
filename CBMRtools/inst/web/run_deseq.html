Run DESeq2 on eset object

Usage

run_deseq(eset, class_id, control, treatment, verbose = TRUE, sortByPValue = TRUE, FDRcutoff = NULL)

Arguments

eset
expression set object with raw count data
class_id
column name in pData(eset) specifying the conditions for differential expression
control
value in treatment specifying the control condition
treatment
value in treatment specifying the positive condition
verbose
add additional information and perform filtering
sortByPValue
whether to sort results by nominal p-value, if verbose = TRUE
FDRcutoff
FDR threshold for filtering results, if verbose = TRUE
control
value in treatment specifying the control condition

Description

Run DESeq2 on eset object
