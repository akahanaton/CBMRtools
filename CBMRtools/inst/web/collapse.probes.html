collapse eSet object by non-unique row feature IDs

Usage

collapse.probes(eset, fdat.id = "SYMBOL", method = c("single.mad", "summarize.mean", "summarize.median", "summarize.max", "summarize.min"))

Arguments

fdat.id
feature id to collapse on
method
= "single" or "summarize" followed by "." followed by "mean", "median", "max" or "min", e.g. "single.mean" "single" chooses one probe for ambiguous probes with maximum of summary function across row, "summarize" aggregates summary metric across multiple rows for each column index
eSet
expression set object

Description

collapse eSet object by non-unique row feature IDs
