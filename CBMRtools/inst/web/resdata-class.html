resdata

Description

Class resdata represents a gene expression microarray data in the Broad '.res' format

Slots

signal
is the [m x n] gene-by-sample matrix of expression levels

calls
is the [m x n] gene-by-sample matrix of P/M/A calls

description
is an m-sized character vector of genes' (or probesets') descriptions

scale
is the scaling factor used with each sample (if any)
